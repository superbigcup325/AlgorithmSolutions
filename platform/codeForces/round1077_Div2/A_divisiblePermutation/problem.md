# A. 可除排列

每个测试的时间限制：1 秒  
每个测试的内存限制：256 MB  
link: https://codeforces.com/contest/2188/problem/A

给你一个整数 \( n \)。构造一个长度为 \( n \) 的排列 \( p \)，满足以下条件：

对于每个 \( 1 \leq i \leq n - 1 \)，都有  
\[|p_i - p_{i+1}| \text{ 能被 } i \text{ 整除}.\]

根据问题的限制条件，可以证明这样的排列总是存在的。

*一个长度为 \( n \) 的排列是一个由 \( 1 \) 到 \( n \) 的 \( n \) 个不同整数按任意顺序组成的数组。例如，\([2, 3, 1, 5, 4]\) 是一个排列，但 \([1, 2, 2]\) 不是排列（因为 2 在数组中出现了两次），\([1, 3, 4]\) 也不是排列（因为 \( n = 3 \) 但数组中出现了 4）。*

## 输入
每个测试包含多个测试用例。第一行包含测试用例的数量 \( t \)（\( 1 \leq t \leq 100 \)）。接下来是每个测试用例的描述。

每个测试用例只有一行，包含一个整数 \( n \)（\( 2 \leq n \leq 100 \)）—— 要构造的排列 \( p \) 的长度。

## 输出
对于每个测试用例，输出 \( n \) 个整数 \( p_1, p_2, \ldots, p_n \)（\( 1 \leq p_i \leq n \)，所有 \( p_i \) 互不相同）—— 你构造的排列。

如果有多个有效的排列，你可以输出其中任意一个。

### 示例

#### 输入
```
2
2
3
```

#### 输出
```
1 2
2 3 1
```

---

## 提示
在第一个测试用例中，\( p = [1, 2] \) 满足条件，因为 \( |p_1 - p_2| = |1 - 2| = 1 \)，它能被 1 整除。

在第二个测试用例中，\( p = [2, 3, 1] \) 满足条件，因为：

- \( |p_1 - p_2| = |2 - 3| = 1 \)，它能被 1 整除，且
- \( |p_2 - p_3| = |3 - 1| = 2 \)，它能被 2 整除。
