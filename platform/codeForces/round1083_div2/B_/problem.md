# B. 西蒙斯与成功蛋糕

每组测试时间限制：1秒  
内存限制：256兆字节

*当我成功时，我们将一起分享蛋糕！*  
— 舜

西蒙斯有 $n$ 个朋友和大量的蛋糕。为了公平地分配蛋糕，你需要帮助他解决以下问题：

- 找到最小的**正**整数 $k$，使得 $n$ 整除 $k^n$。

可以证明，在给定的约束条件下，答案总是存在的。

## 输入
每个测试包含多个测试用例。第一行包含测试用例的数量 $t$ ($1 \leq t \leq 100$)。随后是测试用例的描述。

每个测试用例只有一行，包含一个整数 $n$ ($2 \leq n \leq 10^9$) — 西蒙斯的朋友数量。

## 输出
对于每个测试用例，输出一个整数 — 你找到的最小 $k$。

### 示例

#### 输入
```
4
8
12
369
55635800
```

#### 输出
```
2
6
123
2090
```

---

## 注释
在第一个测试用例中：

- $1^8 = 1$，而 8 不能整除 1；
- $2^8 = 256$，而 8 能整除 256，因为 $256 = 8 \cdot 32$。

因此，最小的可能 $k$ 是 2。

在第二个测试用例中，$12$ 能整除 $6^{12} = 2\,176\,782\,336$，因为 $2\,176\,782\,336 = 12 \cdot 181\,398\,528$。
