# B. 删除排序

每组测试时间限制：1秒  
内存限制：256兆字节

AksLoCoding 正在一个由 $n$ 个正整数组成的数组 $a$ 上玩一个游戏。每轮：

- 如果 $a$ 是**非递减**的，游戏结束。
- 否则，AksLoCoding 可以选择任意一个元素并将其从数组中删除。

请确定游戏结束后，数组中可能剩余的元素的**最小**数量。

---

$a$ 是非递减的，如果对于所有 $1 \leq i \leq m - 1$（其中 $m$ 是数组 $a$ 的长度），都有 $a_i \leq a_{i+1}$。

## 输入
第一行包含一个整数 $t$ ($1 \leq t \leq 1000$)，表示测试用例的数量。

每个测试用例的第一行包含一个整数 $n$ ($1 \leq n \leq 10$)。

每个测试用例的第二行包含 $n$ 个整数，即数组 $a$ 的元素 ($1 \leq a_i \leq 100$)。

## 输出
对于每个测试用例，输出一个整数：数组变为有序后，可能剩余的最小元素数量。

## 示例

### 输入
```
3
4
1 4 2 3
1
100
2
6 7
```

### 输出
```
1
1
2
```

---

## 注释
在第一个测试用例中，可以通过按顺序删除 1、2 和 3，实现剩余 1 个元素的最小值。

在第二个和第三个测试用例中，无法删除任何元素。
