# T727431 [语言月赛 202601] 方程检查

## 题目描述

方程是描述等量关系的式子。在本题中，我们使用小写英文字母 $\texttt{a}-\texttt{z}$ 表示未知量。

千秋是咕洛小学的数学老师，他给学生布置了 $Q$ 道解方程作为家庭作业。现在，他需要检查学生的答案是否正确。

给出每道题目的方程，以及学生的答案，请你检查代入学生的答案后，这个方程是否成立。如果成立，输出 `Yes`，否则输出 `No`。

为了方便描述，我们进一步对本题涉及的方程做出限制：

1. 方程中只包含单个英文字母表示的未知数、不超过 $2^{20}$ 的常数、运算符和等号（`=`）
2. 运算符只包括加法（`+`）、减法（`-`），你无需考虑其他运算。

## 输入格式

第一行为一个正整数 $Q$，表示方程的个数。

接下来，对于每个方程：

- 第一行为一个正整数 $n$，表示该方程中未知量的个数。
- 第二行为一个字符串 $E$，表示方程
- 接下来 $n$ 行，每行一个形如 `a=10`（不含空格） 的字符串，表示一个未知量的值

## 输出格式

输出 $Q$ 行，每行一个字符串，表示该方程是否成立。

## 输入输出样例 #1

### 输入 #1

```
2
2
x+y=25
x=5
y=5
3
a+b=b-c-c
a=2
b=1
c=-1
```

### 输出 #1

```
No
Yes
```

## 说明/提示

对于 $30\%$ 的测试数据，$n=1$。

对于另外 $30\%$ 的测试数据，仅在方程的一侧含有未知量。

对于 $100\%$ 的测试数据，$1 \le Q \le 10$，$1 \le n \le 26$，$1 \le |E| \le 10^5$，其中 $|E|$ 表示字符串 $E$ 的长度。保证未知量的绝对值不超过 $2^{20}$。
